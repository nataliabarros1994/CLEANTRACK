üîß TROUBLESHOOTING
================================================================================

BUILD FAILS:
- Check build logs in Render dashboard
- Verify requirements.txt has all dependencies
- Ensure build.sh is executable (chmod +x build.sh)

APPLICATION ERROR 500:
- Verify all environment variables are set
- Check DATABASE_URL is correctly configured
- View logs: Render Dashboard ‚Üí Logs tab

STATIC FILES NOT LOADING:
- Run in Render Shell: python manage.py collectstatic --clear
- Verify STATIC_ROOT in settings_production.py
- Check WhiteNoise middleware is enabled

DATABASE CONNECTION ERROR:
- Verify DATABASE_URL is from PostgreSQL "Internal Database URL"
- Check database is in same region as web service
- Ensure database is running (not paused)
